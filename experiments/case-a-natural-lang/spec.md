# Case A: 자연어 중심 Spec — Agentic Dynamic Showcase

## 프로젝트 개요
사용자의 스크롤 위치에 따라 필터가 반응하는 지능형 프로젝트 갤러리를 만들어줘.

## 요구사항

### 갤러리 레이아웃
- 프로젝트 카드들을 보기 좋게 그리드로 배치해줘
- 카드에 마우스를 올리면 부드러운 애니메이션을 넣어줘
- 반응형으로 모바일에서도 잘 보이게 해줘

### 스크롤 기반 필터
- 사용자가 스크롤하면 상단에 필터 바가 고정되게 해줘
- 스크롤 위치에 따라 현재 보이는 카테고리가 자동으로 하이라이트되면 좋겠어
- 필터를 클릭하면 해당 섹션으로 부드럽게 이동해줘

### 데이터 로딩
- 데이터를 불러올 때 로딩 상태를 자연스럽게 보여줘
- 스켈레톤 UI 같은 거 넣으면 좋을 것 같아
- 에러가 나면 사용자한테 알려주고 다시 시도할 수 있게 해줘

### 인터랙션
- 카드 클릭하면 상세 정보가 나오게 해줘
- 애니메이션은 부드럽고 자연스러운 느낌으로
- 전체적으로 모던하고 깔끔한 디자인으로 만들어줘

### 기술 스택
- React 기반으로 해줘
- 스타일은 알아서 깔끔하게
- 상태 관리도 적절하게 처리해줘

## 참고사항
- 전반적으로 좋은 UX를 제공해줘
- 성능도 신경 써줘
- 접근성도 고려해줘
